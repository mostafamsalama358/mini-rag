# .gitignore

## What is this?

Tells Git which files **not to track** in the `src/` folder.

## Why does it exist?

You must not commit:

- `.env` (API keys and passwords)
- `__pycache__/` (Python bytecode)
- Virtual environments (`venv/`, `.venv/`)
- Celery beat schedule files
- Uploaded files in `assets/` (see `assets/.gitignore`)

## Important patterns for this project

| Pattern | Why |
|---------|-----|
| `.env` | Secrets |
| `celerybeat-schedule` | Local scheduler state |
| `__pycache__/` | Auto-generated Python cache |

## Where is it used?

- Git when you work inside `src/`
- Works together with `src/assets/.gitignore` for uploaded data

## .NET comparison

Same as `.gitignore` in a .NET solution — ignores `bin/`, `obj/`, and local secrets.
