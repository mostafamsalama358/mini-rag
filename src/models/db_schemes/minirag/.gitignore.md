# .gitignore

## What is this?

Ignores local Alembic files inside the `minirag` DB scheme folder.

## Why does it exist?

These files are personal or generated:

- Real `alembic.ini` (has your DB password)
- Python cache
- Local SQLite or temp DB files

## What it ignores

Typical patterns: `alembic.ini`, `__pycache__/`, `*.pyc`

## Where is it used?

- Git when tracking `src/models/db_schemes/minirag/`
- Works with `docker/.gitignore` which also ignores `minirag/alembic.ini`

## .NET comparison

Like ignoring `appsettings.Local.json` with your real connection string.
